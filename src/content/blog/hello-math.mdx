## Theorems

<Theorem kind="definition" number="1.1" title="Convex function">
A function $f : \mathbb{R}^n \to \mathbb{R}$ is **convex** if for all $x, y$ and all $\lambda \in [0,1]$,
$$
f(\lambda x + (1-\lambda) y) \le \lambda f(x) + (1-\lambda) f(y).
$$
</Theorem>

<Theorem kind="theorem" number="1.2" title="Jensen's inequality">
If $f$ is convex and $X$ is an integrable random variable, then
$$
f(\mathbb{E}[X]) \le \mathbb{E}[f(X)].
$$
</Theorem>

<Theorem kind="proof">
Apply the convexity definition to a finite mixture and take the limit; full details in any standard text on convex analysis.
</Theorem>

## A live visualization

<canvas id="lissajous" width="640" height="320" style="width:100%;border:1px solid var(--border);border-radius:6px;background:var(--bg-elev)"></canvas>

<script is:inline>{`
(() => {
  const c = document.getElementById('lissajous');
  if (!c) return;
  const ctx = c.getContext('2d');
  let t = 0;
  const styles = getComputedStyle(document.body);
  const bg = styles.getPropertyValue('--bg-elev');
  const accent = styles.getPropertyValue('--accent');
  const draw = () => {
    ctx.fillStyle = bg;
    ctx.fillRect(0, 0, c.width, c.height);
    ctx.strokeStyle = accent;
    ctx.lineWidth = 1.5;
    ctx.beginPath();
    const cx = c.width / 2, cy = c.height / 2;
    const A = c.width * 0.38, B = c.height * 0.38;
    const a = 3, b = 2;
    for (let i = 0; i <= 720; i++) {
      const th = (i / 720) * Math.PI * 2;
      const x = cx + A * Math.sin(a * th + t);
      const y = cy + B * Math.sin(b * th);
      if (i === 0) ctx.moveTo(x, y); else ctx.lineTo(x, y);
    }
    ctx.stroke();
    t += 0.01;
    requestAnimationFrame(draw);
  };
  draw();
})();
`}</script>
